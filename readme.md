
### Flash Firmware

1. Connect USB from Mainboard to PC
2. Open Arduino
3. Open Marlin_I3.ino
4. Set board under `Tools` menu to `Arduino Mega 2560 or Mega ADK`
5. Choose USB serial port
6. Upload firmware to board `File -> Upload (CTRL+U)`
